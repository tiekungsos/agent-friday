# Friday Jarvis Roadmap

## Vision
สร้าง Friday ให้กลายเป็นผู้ช่วยสไตล์ JARVIS บน MacBook ที่คุยได้ คิดได้ จำได้ และลงมือทำงานให้บอสได้จริงผ่าน native desktop experience

## Current Status
- Phase 4 complete
- Electron app พร้อม command center UI
- local memory / notes / workspace storage
- planner + reviewer loop ระดับเริ่มต้น
- tool bench และ native settings
- tray + shortcut + packaging config

## Roadmap

### Phase 5 — Native operator foundations
**Goal:** ให้ Friday เริ่มลงมือทำงานจริงบนเครื่อง Mac ได้มากขึ้น

- Native speech-to-text / text-to-speech flow
- Calendar / reminders integration
- Browser automation starter tools
- Import / export workspace
- Build macOS `.app` pipeline บนเครื่อง Mac จริง

### Phase 6 — Multi-agent execution
**Goal:** ให้ agent team ทำงานหลายรอบอย่างมี state

- planner → executor → reviewer loop จริง
- task queue และ step status
- mission history พร้อม replay
- richer tool routing ต่อ agent แต่ละตัว

### Phase 7 — Personal chief-of-staff
**Goal:** ทำให้ Friday ช่วยงานประจำวันได้ลึกขึ้น

- smart daily briefing
- meeting prep / follow-up
- document drafting and memory recall
- personalized command suggestions

## Priority Backlog
1. Native macOS packaging and release checklist
2. Calendar + reminders tools
3. Browser automation MVP
4. Structured mission history
5. Real multi-agent execution loop

## Success Metrics
- บอสเปิดแอปแล้วสั่งงานได้ภายใน 1 นาที
- Mission หนึ่งชุดสามารถสร้าง plan + ใช้ tools + เก็บ memory ได้ครบ
- build `.app` สำเร็จบน Mac จริง
- มี issue backlog ที่พร้อมลุยต่อเป็นลำดับ
